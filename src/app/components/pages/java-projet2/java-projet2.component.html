Étudiant Management Système

  • 2024

C'est l'un de mes projets Spring Boot. Il simule un simple gestionnaire de liste d'étudiants. Où je peux ajouter des informations sur les étudiants, modifier des informations ou supprimer des étudiants.

Détail du projet

J'utilise Spring Suite Tool 4 pour développer ce projet, avec des dépendances : Spring Web, Spring Data JPA, MySQL Driver, Thymeleaf, Dev Tools. J'utilise l'application MySQL Workbench comme lien vers la base de données du projet. Dans le projet, j'ai utilisé la classe Student (Entity) pour définir les champs d'information et l'ID nécessaires. J'ai créé StudentRepository implémentant le référentiel JPA afin de pouvoir utiliser des ressources JPA utiles.
Ensuite, j'ai les classes StudentService (Service) et StudentController (Controller) pour gérer les services complexes et la logique de traitement des données, gérer les demandes des utilisateurs et naviguer dans le programme, en renvoyant la réponse correspondante.
J'ai créé 3 pages HTML supplémentaires comme interface de mon site Web, en utilisant boostrap CSS pour le styler. Un petit ajustement dans application.properties (connexion à la base de données et ajout d'hibernate auto ddl) et la section principale (implémente CommandLineRunner) et j'ai pu démarrer le site Web du projet sur le port localhost 8080.

Avantages du projet

  • Utiliser l'API et établir un lien avec le Front End
  • Belle interface

Conclusion

Mais comme le programme ne s'exécute que localement et par intermittence, lorsque j'éteigne le programme, toutes les données seront perdues.
