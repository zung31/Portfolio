Interface d'une site Smart Home

  • 2023

Le projet Smart Home est une plateforme en ligne multifonctionnelle qui offre aux utilisateurs la capacité de contrôler et de surveiller les appareils de leur maison intelligente. En utilisant une interface intuitive et pratique, les utilisateurs peuvent accéder aux différentes pièces et appareils tels que l'éclairage, les stores, les capteurs de température, les caméras de sécurité et bien d'autres. L'objectif du projet est de fournir une gestion intelligente des appareils domestiques, offrant ainsi commodité et sécurité aux utilisateurs.​
J'utilise les langages de programmation: HTML, CSS, JavaScript, PHP, un peu de Python pour cet interface

Conclusion

C'est le premier projet sur lequel je travaille avec ces langages, donc l'interface est encore limitée et n'a pas beaucoup de fonctionnalités (pas de backend, juste quelques fichiers php connectés à BDD pour sauvegarder les données).
