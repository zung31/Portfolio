Serpent Game

  • 2024

C'est l'un de mes projets Java. Cela m'a aidé à me familiariser avec des bibliothèques comme JFrame, Jpanel, me permettant d'afficher un simple écran de jeu de pomme mangeuse de serpent.

Comment contrôler le serpent?

Dans le programme, j'ai programmé un serpent coloré, d'une longueur par défaut de 6, dont la taille augmentera de 1 unité après avoir mangé une pomme. La pomme apparaîtra à un endroit aléatoire, après avoir été mangée, elle disparaîtra et apparaîtra à un autre endroit.
Au-dessus de l'écran se trouve Score pour mettre à jour le score actuel du joueur. Le jeu se terminera lorsque la tête du serpent entrera en collision avec le mur ou le corps du serpent.
En parallèle, j'ai également programmé le Key Adapter pour limiter les pas du joueur (par exemple, lorsque le serpent se déplace vers la droite, il ne peut pas aller vers la gauche).
Si vous voyez le serpent bouger trop vite, on peut ajuster la constante DELAY pour changer sa vitesse.

Avantages du projet

  • Le programme est également assez complet car il prend en compte tous les scénarios possibles
  • On peut personnaliser le serpent à son intérêt

Conclusion

Pourtant, le programme est même que la version de base qui était crée depuis longtemps. J'espère pouvoir améliorer l'interface et rendre le programme plus intéressant à l'avenir.
